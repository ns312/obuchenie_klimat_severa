# Интерактивный Портал Обучения Менеджеров «Климат Севера»

Стильное и современное веб-приложение (Single Page Application) для обучения менеджеров по холодным звонкам с использованием скрипта **«Эксперт-Помощник»** и 10-дневной программы интеграции с **AMO CRM**.

## 🚀 Возможности Портала

1. **Многопользовательская авторизация**:
   - Менеджеры регистрируются со своим email и паролем. Прогресс каждого сотрудника сохраняется изолированно в `localStorage` браузера.
2. **5 уровней интерактивного обучения (Дни 1-10)**:
   - **Уровень 1**: Изучение болей клиентов (кафе, рестораны, банки, офисы). Интерактивная игра на сопоставление.
   - **Уровень 2**: Скрипт «Эксперт-Помощник». Диалоговый тренажер звонка с ветвлением ответов и симуляцией реального разговора.
   - **Уровень 3**: Возражения («У нас свой мастер», «Нам не актуально» и др.). Переворачивающиеся карточки возражений и тест на закрепление.
   - **Уровень 4**: Ведение AMO CRM. Симулятор канбан-доски с драг-энд-дропом сделок и планированием задач по регламенту.
   - **Уровень 5**: Дефектные акты, дожим в WhatsApp и итоговый экзамен.
3. **Панель Управления Куратора**:
   - Специальный защищенный интерфейс для администратора.
   - Вход под реквизитами:
     - **Email**: `sultan.marketing.dubai@gmail.com`
     - **Пароль**: `312312312`
   - Просмотр детальной статистики успеваемости по каждому менеджеру.
   - Удобный просмотр отправленных ответов на домашние задания с возможностью **принять** или **отклонить** работу.
   - Выгрузка результатов в JSON отчет.

## 🛠️ Стек технологий

- HTML5 (Семантическая структура)
- CSS3 (Vanilla CSS, HSL переменные, стеклянный интерфейс Glassmorphism, 3D перевороты карточек, адаптивный грид/флекс)
- Modern JavaScript (ES6+, LocalStorage API, Drag & Drop API, управление динамическим DOM-деревом)

## 💻 Локальный запуск

1. Склонируйте репозиторий:
   ```bash
   git clone <URL_ВАШЕГО_РЕПОЗИТОРИЯ>
   cd sales-trainer
   ```
2. Просто откройте файл `index.html` в любом браузере (двойной клик на файл) или запустите легкий локальный сервер:
   ```bash
   python3 -m http.server 8000
   ```
   После чего перейдите по адресу `http://localhost:8000`.
